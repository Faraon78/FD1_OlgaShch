‹ SUP › и ‹ SUB › верхний и нижний индексы — парные теги

Нижние индексы могут применяться, например, в написании химических формул:
H2SO4, С2H5OH
Верхние индексы нужны в математике:
x2 + y3 + 72 = z4
Так же их можно использовать в написании времени: 910— 1715
Можно поиграть с вложенностью этих тегов: Bzzzzzz Упссссс
Или их сочетанием
ВОТ ТАКАЯ ЗАДАЧА

‹ PRE › предварительно форматированный текст — парный тег

Такой тег сохнаняет
    все пробелы и переносы
        введенные пользователем
            для создания визуальных
                   эффектов.

‹ ADRESS › для написания адреса — парный тег

Текст внутри будет по умолчанию с курсивным начертанием, а тег рекомендуется использовать для
последующей корректной работы поисковых систем, собирающих данные об авторах сайта.

‹ BLOCKQUOTE › для написания цитат в тексте — парный тег

Тег удобен своим форматированием:

Цитата будет выделена отступами слева и справа,
можно так писать стихи.

‹ DL ›, ‹ DT ›, ‹ DD › — парные теги

Теги предназначены для создания списка определений. Каждый такой список начинается с контейнера ‹dl›,
куда входит тег ‹dt› создающий термин и тег ‹dd› задающий определение этого термина. Например:

Бухгалтерский учет
упорядоченная система сбора, регистрации и обобщения информации в денежном выражении о состоянии имущества,
обязательствах и капитале организации и их изменениях путём сплошного, непрерывного и документального
отражения всех хозяйственных операций.

‹ KBD › — парный тег

Используется для обозначения текста, который набирается на клавиатуре или для названия клавиш.
Браузеры обычно помечают текст в контейнере ‹kbd› моноширинным шрифтом.
Например, для сохранения текста программы я регулярно нажимаю сочетание клавиш CTRL + S.
Вероятно, название тега произошло от слова Keyboard - клавиатура.

‹ VAR › — парный тег

Используется для выделения переменных компьютерных программ.
Браузеры обычно помечают текст в контейнере ‹var› курсивным начертанием.
Например, в программах часто используют переменную name

‹ Q › — парный тег

Как и тег ‹blockquote › тег ‹q› используется для написания цитат, он не делает отступы, но помещает цитату в кавычки.
Моя любимая поговорка: Глаза боятся, а руки делают.

‹ INS › и ‹ DEL › — парные теги

Применяются для выделения добавленного и удаленного текстов в новой версии документов.
Сегодня я очень устала мне было очень интересно.

Уведомляет браузер отображать текст без переносов в противовес одиночному тегу ‹br›, который ставит переносы строк по желанию разработчика.
