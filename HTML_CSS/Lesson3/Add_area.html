Teг <AREA>

Тег <area> (от англ. area - область) определяет активные области изображения, которые являются ссылками.

Рисунок с привязанными к нему активными областями называется в совокупности картой-изображением. Такая карта по внешнему виду ничем не отличается от обычного изображения, но при этом оно может быть разбито на невидимые зоны разной формы, где каждая из областей служит ссылкой. Элемент <area> задаёт форму области, её размеры, устанавливает адрес документа, на который следует сделать ссылку. <area> всегда располагается в контейнере <map>, который связывает координаты областей с изображением.

Внутри тега <map> создаются теги <area>, каждый из которых задает интерактивную область, являющуюся ссылкой.

В данном файле к тегам <area> были использованы следующие атрибуты:

  • Атрибут Shape указывает форму выделяемой области и может принимать значения:
    • default - вся область
    • rect - прямоугольник (задаются координаты X1, Y1, X2, Y2)
    • circle - круг (задаются координаты центра X, Y и радиус круга)
    • poly - многоугольная область (задаются координаты всех вершин x1,y1,x2,y2,...,xn,yn)
  • Атрибут Coords задает координаты необходимой области
  • Атрибут Href определяет гиперссылку на указанной области
  • Атрибут Target указывает в каком окне браузера ссылку
  • Атрибут alt определяет альтернативный текст ссылки
Внутри тега <Img> был задан атрибут usemap="#language" для связи изображения с тегом <map>
Форма активных областей - круг.

Определенные проблемы вызвало определение координат X и Y для задания атрибута coords. Для решения этой проблемы в HTML код был вставлен код JavaScript (скопирован из Интернета), отображающий на экране координаты курсора.
После определения всех координат внутри рисунка JavaScript был закомментирован, но не удален из программы.

Также изначально при выделении области на картинке был виден круг (сама активная область), чтобы ее скрыть в стилях использовано свойство outline-color: transparent.
