Кодирование цвета в теге <Font>

Если в устаревшем теге <Font> указать название цвета каким-нибудь словом(словосочетанием), то браузер это название преобразует в цвет.

Например, цвет BruceWillis или цвет Candy, а можно задать цвет ShchahliakOlga и он тоже отобразится. Как же это работает?

Все вводимые символы программа пытается преобразовать в 16-ричный код: если символ соответствует 16-ричной системе (0-9 A-F), то он остается, если не соответствует, то вместо него подставляется 0.

Так, #BruceWillis будет преобразован в B00CE000000. Так как в системе RGB закодированы 3 цвета, то количество знаков в полученном коде увеличивается до числа, кратного 3. В коде B00CE000000 11 знаков, система подставит в него еще один 0 чтобы получилось 12 знаков: B00CE0000000. Далее код делится на 3 части: B00C E000 0000. На этом этапе значения RGB усекаются по отдельности. Если отдельные значения RGB имеют длину более 8 символов, они усекаются до 8 символов путем удаления символов слева. Если длина отдельных значений RGB составляет менее 8 символов, они обрезаются путем удаления символов справа. Наш код B00C E000 0000 усекается до: B0 E0 00. Это и будет цветовым кодом для браузера.

Теперь проверим: цвет #BruceWillis и цвет #B0E000 — они идентичны.
Цвет Candy будет преобразован в код #CA0d00

Если в результате преобразования кода получается, что блоки цветов начинаются с 0, то система обрежет нули слева, пока хотя бы один блок будет начинаться не с 0. Это правило можно проверить на преобразовании цвета ShchahliakOlga:
ShchahliakOlga → 00C0A000A0000A → 00C0A000A0000A0 → 00C0A 000A0 000A0 → C0A 0A0 0A0 → C0 0A 0A.

Проверим: цвет ShchahliakOlga и цвет #C00A0A — также идентичны.

Безопасная палитра

При оформления в интернете главная задача — это корректная передача цветов для разных мониторов и типов компьютеров, для различных браузеров. Если браузер не может по каким-то причинам передать указанный цвет, он подбирает похожий либо смешивает несколько рядом стоящих цветов. Именно поэтому согласованный дизайн иногда может визуально отличаться для разных пользователей при просмотре веб-сайта.

Для того, чтобы передать пользователям правильный цвет, была составленная таблица цветов, которые ввиду особенности их цветового кода отображаются одинаково правильно на всех платформах и во всех браузерах. Эту таблицу цветов назвали «Безопасной палитрой» или Web Safe. Она состоит из 216 цветов. Конечно, это немного для web-дизайнера, но единого стандарта всей цветовой палитры так и не создано.

Если Ваш сайт имеет какой-либо цветовой фон, то лучше для этой цели использовать именно Web Safe цвета. Это гарантирует Вам, что цвет не будет смущать пользователя при отображении на разных компьютерных платформах. Даже старые комьпьютеры, которые все еще активно используются, будут представлять ресурс точно, без колебаний и различий в цветах.
